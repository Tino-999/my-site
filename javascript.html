JavaScript Web APIs HTTP CORS
Wo diese Dinge eigentlich leben

JavaScript ist nur eine Sprache. Viele Dinge, die man im Alltag benutzt, gehören eigentlich zur Umgebung des Browsers oder zum Web-Protokoll.

JavaScript → Programmiersprache
Web APIs → Funktionen des Browsers (fetch, Response, Headers)
HTTP → Kommunikationsprotokoll des Webs
CORS → Sicherheitsregel im Browser für API-Zugriffe

Ein Programm ist nur Namen, Handlungen, Entscheidungen, Zeit, Struktur und Zusammenarbeit.
